Overview

The UnitMeasurements::UnitGroup class provides a collection of units with methods to retrieve units by name, check if a unit is defined, and more. It serves as a container for organizing and working with units in a unit group.

def unit_for!(name)
  unit = unit_for(name)
  raise UnitError, name unless unit

  unit
end

def unit_with_name_and_aliases
  units.each_with_object({}) do |unit, hash|
    unit.names.each { |name| hash[name.to_s] = unit }
  end
