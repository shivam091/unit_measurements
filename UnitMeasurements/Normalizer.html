.normalize(string) ⇒ String

Normalizes a string containing special symbols, exponents, fractions, or ratios.

The normalize method in the UnitMeasurements::Normalizer module processes the input string to replace special symbols with their equivalent representations, such as converting superscript numbers to regular numbers, special fraction symbols to their numeric fractions, and ratios to fractional format.

Examples:

UnitMeasurements::Normalizer.normalize("¾")
=> "3/4"

UnitMeasurements::Normalizer.normalize("1:2")
=> "1/2"

def normalize(string)
  string.dup.tap do |str|
    if str =~ Regexp.new(EXPONENT_REGEX)
      EXPONENTS_SYMBOLS.each do |search, replace|
        str.gsub!(search) { "#{replace}" }
      end
    end

    str.gsub!(FRACTION_REGEX) { " #{FRACTIONS_SYMBOLS[$1]}" }

    str.gsub!(RATIO_REGEX)    { "#{$1.to_i}/#{$2.to_i}" }

    str.strip!
  end
end
